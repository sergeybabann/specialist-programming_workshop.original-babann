----- Использование веб-сервера browser-sync в IntelliJ IDEA -----

- 1) Установить веб-браузер browser-sync глобально:

npm install -g browser-sync

- 2) Из корневого каталога проекта запустить команду в терминале:
browser-sync start --server --files "*,**/*"

Эта команда дает задание веб-серверу отслеживать изменения во всех файлах html, css, проекта
